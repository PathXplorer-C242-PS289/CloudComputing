# Getting Started

initiate ```node_modules```
```bash
npm install
```

start project
```bash
npm start
```

create .ENV for database connection
```bash
// db connection
DB_HOST=DB_HOST
DB_USER=DB_USER
DB_NAME=DB_NAME 
# DB_PASS=DB_PASS
// optional
JWT_SECRET=YOUR_JWT_SECRET_KEY
// gmail connection
EMAIL_USER=YOUR_EMAIL_USER
EMAIL_PASS=YOUR_EMAIL_PASS
```

## Require
- mysql databse
